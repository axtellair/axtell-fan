# AXTELL FAN — Vue 3 Demo

This is a scaffolded Vue 3 + Vite project generated from an existing static `index.html`.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

The app mounts to `#app`. Styles were extracted to `src/assets/styles/main.css` and components live under `src/components/`.
